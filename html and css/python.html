The Python Folder

Okay, in the python folder you will see a ff invasion folder and a work folder.
The ff invasion folder holds the components of an alien invasion game that is similar to Space Invaders. The blue print of this game was given in a book, 'Python Crash Course' by Eric Matthes. This is the book I use to teach my programming class. I made some modifications and this is what you see. To run the game you must run the alien_invasion.py file.
In my class, I will at times hold a game club depending on their progress. The program in the work folder ramdomly selects an opponent and makes sure that everyone must play before one can play again.
